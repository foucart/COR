Globally Optimal Estimation Functionals: Accurate Scenario

One considers first the global setting, where the performance of an estimation functional is assessed, in the absence of observation errors, through
One starts by determining the minimal global worst-case error over all estimation functionals, computed according to Theorem 6. Recall that this relies on the fact that quadratric estimation functionals are optimal, as established in Theorem 2. This fact is also corroborated empirically by verifying the numerical agreement with a general lower bound, computed according to Proposition 8.
% minimization over quadratic functionals
cvx_begin quiet
variable cp nonnegative
variable cpp nonnegative
variable B(m,m) symmetric
minimize .5*(cp+cpp)
subject to
cp*T'*T - A + L'*B*L == semidefinite(N);
cpp*T'*T + A - L'*B*L == semidefinite(N);
cvx_end
gwce_min = cvx_optval;
% determination of the lower bound
cvx_begin quiet
variable c1 nonnegative
variable c2 nonnegative
variable d
minimize .5*(c1+c2)
subject to
[c1*T'*T - A + d*L'*L, -d*L'*L;...
-d*L'*L, c2*T'*T + A + d*L'*L]...
== semidefinite(2*N);
cvx_end
lb = cvx_optval;
fprintf(['The minimal global worst-case error'...
' and the general lower bound agree.\n' ...
'They are (up to numerical errors) %.3f and %.3f, respectively.'],...
gwce_min,lb)
The minimal global worst-case error and the general lower bound agree. They are (up to numerical errors) 136.499 and 136.505, respectively.

Function computing the locally optimal recovery map

The function defined below computes the Chebyshev center and radius according to Theorem 1, whose statement involves the orthogonal projector P onto and the element minimizing subject to .
function [cheb_center, cheb_radius] = cheb(A,a,alp,T,L,y)
% auxiliary quantities
N = length(a);
invT = inv(T);
P = eye(N) - L'*inv(L*L')*L;
fy = (invT*invT')*L'*inv(L*(invT*invT')*L')*y;
% solve the first SDP
cvx_begin quiet
variable c nonnegative
variable d
minimize d
subject to
[P*(c*(T'*T) - A)*P,...
-P*(A*fy+a); -(A*fy+a)'*P,...
d - c*(1-norm(T*fy)^2) - fy'*A*fy - 2*a'*fy - alp]...
== semidefinite(N+1);
cvx_end
sig = cvx_optval;
% solve the second SDP
cvx_begin quiet
variable c nonnegative
variable d
maximize d
subject to
[P*(c*(T'*T) + A)*P,...
P*(A*fy+a); (A*fy+a)'*P,...
-d - c*(1-norm(T*fy)^2) + fy'*A*fy + 2*a'*fy + alp]...
== semidefinite(N+1);
cvx_end
iot = cvx_optval;
cheb_radius = (sig-iot)/2;
cheb_center = (sig+iot)/2;
end
