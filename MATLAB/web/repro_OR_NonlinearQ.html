cvx_begin quiet
variable e_opti
variable e1(I_size)
variable e2(I_size)
variable c(m,I_size)
variable sig(I_size,I_size) nonnegative
variable s(L_size,I_size) nonnegative

minimize e_opti
subject to
sum(sig) == 1;

% introduce the model set
N = 50;
n = 20;
V = rand(N,n);
P = eye(N) - V*inv(V'*V)*V';
epsilon = 0.7;
g_aux = rand(N,1);
g = (epsilon/2)*g_aux/norm(P*g_aux);
% introduce the observation process
m = 30;
U = randn(N,m);

% introduce the functional to be estimated

Theorem 1 reveals that an optimal estimation functional has a sup-affine form. It is determined by solving the following second-order cone program stemming from Proposition 5.
cvx_begin quiet
variable e_opti
variable e1(I_size)
variable e2(I_size)
variable c(m,I_size)
variable sig(I_size,I_size) nonnegative
minimize e_opti
subject to
sum(sig) == 1;
for i=1:I_size
e1(i)+e2(i) <= 2*e_opti;
V' * (W(:,i)-U*c(:,i)) == 0;
V' * (-W*sig(:,i)+U*c(:,i)) == 0;
epsilon*norm(W(:,i)-U*c(:,i)) <= ...
e1(i) - g'*(W(:,i)-U*c(:,i));
epsilon*norm(-W*sig(:,i)+U*c(:,i)) <= ...
e2(i) - g'*(-W*sig(:,i)+U*c(:,i));
end
cvx_end
fprintf(['The minimal error over all estimation functionals '...
'is equal to %.3f.'],e_opti)
