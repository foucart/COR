Globally Optimal Estimation Functionals: Accurate Scenario

One considers first the global setting, where the performance of an estimation functional is assessed, in the absence of observation errors, through
One starts by determining the minimal global worst-case error over all estimation functionals, computed according to Theorem 6. Recall that this relies on the fact that quadratric estimation functionals are optimal, as established in Theorem 2. This fact is also corroborated empirically by verifying the numerical agreement with a general lower bound, computed according to Proposition 8.
% minimization over quadratic functionals
cvx_begin quiet
variable cp nonnegative
variable cpp nonnegative
variable B(m,m) symmetric
minimize .5*(cp+cpp)
subject to
cp*T'*T - A + L'*B*L == semidefinite(N);
cpp*T'*T + A - L'*B*L == semidefinite(N);
cvx_end
% determination of the lower bound
gwce_min = cvx_optval;
cvx_begin quiet
variable c1 nonnegative
variable c2 nonnegative
variable d
minimize .5*(c1+c2)
subject to
[c1*T'*T - A + d*L'*L, -d*L'*L;...
-d*L'*L, c2*T'*T + A + d*L'*L]...
== semidefinite(2*N);
cvx_end
lb = cvx_optval;
fprintf(['The minimal global worst-case error'...
' and the general lower bound agree.\n' ...

In the presence of deterministic observation errors with , the global worst-case error of an estimation functional is adjusted to become
Theorem 7 guarantees that the minimum over all estimation functionals is again achieved for a quadratic functional and that it can still be computed by solving a semidefinite program. Note that it has to be larger than the minimal global worst-case error in the accurate setting.
U = eye(m);
eta = 0.01;
cvx_begin quiet
variable cp1 nonnegative
variable cp2 nonnegative
variable cpp1 nonnegative
variable cpp2 nonnegative
variable B(m,m) symmetric
minimize .5*(cp1+cpp1+eta^2*(cp2+cpp2))
subject to
[cp1*T'*T - A + L'*B*L, L'*B;...
B*L, cp2*U'*U + B]...
== semidefinite(N+m);
[cpp1*T'*T + A - L'*B*L, -L'*B;...
-B*L, cpp2*U'*U - B]...
== semidefinite(N+m);
cvx_end

a = rand(N,1)/N;
alp = rand;
T = eye(N)/sqrt(N);
f0_aux = randn(N,1);
f0 = 0.9*f0_aux/norm(T*f0_aux);
y = L*f0_aux;
P = eye(N) - L'*inv(L*L')*L;
fy = (invT*invT')*L'*inv(L*(invT*invT')*L')*y;
The minimal local worst-case error is computed, along with the Chebyshev center, by solving two semidefinite programs.
% solve the first SDP
cvx_begin quiet
variable c nonnegative
variable d
minimize d
subject to
[P*(c*(T'*T) - A)*P,...
-P*(A*fy+a); -(A*fy+a)'*P,...
d - c*(1-norm(T*fy)^2) - fy'*A*fy - 2*a'*fy - alp]...
== semidefinite(N+1);
cvx_end
sig = cvx_optval;
% solve the second SDP
cvx_begin quiet
variable c nonnegative
variable d
maximize d
subject to
[P*(c*(T'*T) + A)*P,...
P*(A*fy+a); (A*fy+a)'*P,...
-d - c*(1-norm(T*fy)^2) + fy'*A*fy + 2*a'*fy + alp]...
== semidefinite(N+1);
cvx_end
iot = cvx_optval;
lwce = (sig-iot)/2;
center = (sig+iot)/2;
fprintf(['The minimal local worst-case error is %.3f,\n'...
'while the Chebyshev center is %.3f.'], lwce, center)
