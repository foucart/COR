INTERSECTION OF 2 HYPERELLIPSOIDS

Setting the problem

close all; clear;
% dimension of the ambiant Hilbert space H
N = 20;
% two Hilbert-valued linear maps defined on H
R = randn(N,N);
S = randn(N,N);
% dimension of a Hilbert space Z being the range of the quantity of interest Q
n = 10;
% quantity of interest
Q = randn(n,N);
% number of observations
m = 7;
% observation map
Lam = randn(m,N);
% construct a basis for ker(Lam)
[U_aux,~] = qr(Lam');
% the columns of U are an ONB for ker(Lam)
U = U_aux(:,m+1:N);

Computation of the radius of information (Theorem 1)

cvx_begin quiet
warning('off')
variable a nonnegative
variable b nonnegative
minimize a+b
subject to
U'*(a*R'*R+b*S'*S-Q'*Q)*U == semidefinite(N-m);
% The symmetric matrices R', S', and Q' defined in Eq.(11) of the paper
% are computed here as U'*R'*R*U, U'*S'*S*U, and U'*Q'*Q*U, respectively.
cvx_end
rad = sqrt(cvx_optval);
fprintf(['For the two-hyperellipsoid-intersection model set,\n' ...
'the radius of information of the observation map' ...
' for the estimation of Q is %.3f.'],rad)
For the two-hyperellipsoid-intersection model set, the radius of information of the observation map for the estimation of Q is 2.281.

Verification of the side result (Proposition 6)

% map T_N defined in the proof
TN = U'*(a*R'*R+b*S'*S)*U;
TN_aux = inv(sqrtm(TN));
[v,~]=eigs(TN_aux*U'*Q'*Q*U*TN_aux,1);
h_aux = U*TN_aux*v;
h = h_aux/norm(R*h_aux);
% the norm of Rh and Sh should both be =1
[norm(R*h) norm(S*h)]
ans = 1×2
1.000000000000000 0.999998356687224
% the norm of Qh should equal the radius of information
[norm(Q*h) rad]
ans = 1×2
2.280953053550632 2.280956020945010

TWO-SPACE PROBLEM

following Section 3.1

Setting of the problem

close all; clc;
% dimension of the ambiant Hilbert space H
N = 20;
% two orthogonal projectors defined on H
n1 = 5;
V = rand(N,n1);
PV = eye(N) - V*inv(V'*V)*V';
n2 = 3;
W = rand(N,n2);
PW = eye(N) - W*inv(W'*W)*W';
% approximability parameters
eps = 0.2;
eta = 0.8;
% dimension of a Hilbert space Z being the range of the quantity of interest Q
n = 10;
% quantity of interest
Q = randn(n,N);
% number of observations
m = 7;
% observation map
Lam = randn(m,N);
% construct a basis for ker(Lam)
[U_aux,~] = qr(Lam');
% the columns of U are an ONB for ker(Lam)
U = U_aux(:,m+1:N);

Computation of the radius of information (Theorem 7)

cvx_begin quiet
variable c nonnegative
variable d nonnegative
minimize c*eps^2 + d*eta^2
subject to
U'*(c*PV + d*PW - Q'*Q)*U == semidefinite(N-m)
cvx_end
rad = sqrt(cvx_optval);
fprintf(['For the two-space model set,\n' ...
'the radius of information of the observation map' ...
' for the estimation of Q is %.3f.'],rad)
For the two-space model set, the radius of information of the observation map for the estimation of Q is 2.334.

L1-INACCURATE DATA

following Section 4

Case 1: Small η

Setting the problem

close all; clc;
% dimension of the ambiant Hilbert space H
N = 20;
% Hilbert-valued linear map defined on H
R = randn(N,N);
% approximability parameter
eps = 0.5;
% dimension of a Hilbert space Z being the range of the quantity of interest Q
n = 10;
% quantity of interest
Q = randn(n,N);
% number of observations
m = 7;
% observation map
Lam = randn(m,N);
% pseudo-inverse of Lam
Lamdag = Lam'*inv(Lam*Lam');
% uncertainty parameter
eta = 0.01;

Preparations for the calculations to come

We introduce linear maps defined on the extended Hilbert space as in Eq.(38) of the paper.
% augmented observation map Gam
Gam = [Lam, zeros(m,1)];
% construct a basis for ker(Gam)
[U_aux,~] = qr(Gam');
% the columns of U are an ONB for ker(Gam)
U = U_aux(:,m+1:N+1);
% augmented Hilbert-valued linear map S
S = [zeros(1,N), 1/eta];
% augmented Hilbert-valued linear maps R^i and Q^i
R_aug = zeros(N,N+1,m);
Q_aug = zeros(n,N+1,m);
for i=1:m
ui = Lamdag(:,i);
% the i-th elements in the tensors R_aug and Q_aug represent R^i and Q^i, respectively
R_aug(:,:,i) = 1/eps*[R, -R*ui];
Q_aug(:,:,i) = [Q, -Q*ui];
end

Compute the lower bounds

lb_p = zeros(m,1);
% array to store the extremizers a and b
ab = zeros(m,2);
for i=1:m
Ri = R_aug(:,:,i);
Qi = Q_aug(:,:,i);
cvx_begin quiet
variable a nonnegative
variable b nonnegative
minimize a+b
subject to
U'*(a*Ri'*Ri + b*S'*S - Qi'*Qi)*U == semidefinite(N+1-m);
cvx_end
lb_p(i) = sqrt(cvx_optval);
ab(i,:) = [a,b];
end

Construction of the regularizers

Gamdag = Gam'*inv(Gam*Gam');
Del = zeros(n,m,m);
for j=1:m
aj = ab(j,1);
bj = ab(j,2);
Rj = R_aug(:,:,j);
Qj = Q_aug(:,:,j);
aux = Gamdag - ...
U*inv(U'*(aj*Rj'*Rj+bj*S'*S)*U)*U'*(aj*Rj'*Rj+bj*S'*S)*Gamdag;
Del(:,:,j) = Qj*aux;
end

Verification of the sufficient condition

[~,k] = max(lb_p);
Mk = zeros(m,1);
for i=1:m
Ri = R_aug(:,:,i);
Qi = Q_aug(:,:,i);
aux = Qi - Del(:,:,k)*Gam;
cvx_begin quiet
variable a nonnegative
variable b nonnegative
minimize a+b
subject to
a*Ri'*Ri+b*S'*S - aux'*aux == semidefinite(N+1);
cvx_end
Mk(i) = sqrt(cvx_optval);
end
% sufficient condition: the maximum of Mk should be equal to lb_p(k)
[max(Mk) lb_p(k)]
ans = 1×2
1.352686257969239 1.352686257878223

Verification of the side results (Proposition 14)

cvx_begin quiet
variable gam nonnegative
variable a(m) nonnegative
variable b(m) nonnegative
variable Del_lin(n,m) % a linear recovery map treated as a variable in the SDP
minimize gam
subject to
a+b <= gam
for i=1:m
aux = Q_aug(:,:,i) - Del_lin*Gam;
Ri = R_aug(:,:,i);
[eye(n), aux; aux', a(i)*Ri'*Ri+b(i)*S'*S] == semidefinite(N+n+1)
end
cvx_end
% the optimal value of this SDP should coincide with the maxima of Mk and lb_p(k),
% confirming that there is a linear recovery map with minimal global worst-case error
% when eta is small enough
[max(Mk) lb_p(k) sqrt(cvx_optval)]
ans = 1×3
1.352686257969239 1.352686257878223 1.352686257837929
fprintf(['The norm of difference between linear map obtained by regularization \n' ...
'and the linear map obtained as solution of the SDP is %.3f,\n' ...
'illustrating once again the nonuniqueness of optimal recovery maps.'], norm(Del(:,:,k)-Del_lin))
The norm of difference between linear map obtained by regularization and the linear map obtained as solution of the SDP is 0.360, illustrating once again the nonuniqueness of optimal recovery maps.
